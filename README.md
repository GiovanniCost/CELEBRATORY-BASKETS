<h1> CESTAS DE PRESENTES - E-commerce de Cestas Comemorativas</h1>
<br>
<h2>Um site elegante e funcional desenvolvido para a venda de cestas personalizadas (café da manhã, datas comemorativas e brindes corporativos). O projeto foca em uma experiência de compra fluida e design responsivo</h2>
<br>
<h2>Tecnologias e Ferramentas</h2>
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JAVASCRIPT</li>
</ul>
<br>
<h2> Principais Funcionalidades</h2>
<ul>
  <li>Catálogo Dinâmico: Visualização de diferentes tipos de cestas com preços e descrições.</li>
  <li>Lógica de filtragem por categoria (ex: "Aniversário", "Romântica")</li>
  <li>Navegação otimizada para desktops, tablets e smartphones.</li>
</ul>
